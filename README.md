# MyColor Raycast Extension

A minimal Raycast extension that allows users to quickly access and copy their favorite RGB colors from a predefined list.

## Features

- Quick access to favorite RGB colors through Raycast
- One-click copying of RGB values to clipboard
- Visual color previews in the list interface
- Local storage of color data (no network dependencies)

## Development

### Prerequisites

- Node.js 18+
- npm or yarn
- Raycast (for testing)

### Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

### Scripts

- `npm run dev` - Start development mode
- `npm run build` - Build the extension
- `npm run lint` - Run ESLint
- `npm run fix-lint` - Fix ESLint issues
- `npm run type-check` - Run TypeScript type checking
